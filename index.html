El MCP Server de DocuSign expone estas herramientas que cualquier agente de IA puede invocar vía OAuth 2.0.

🔍search_agreementsNavigator
Busca contratos por nombre, estado, fecha o metadata usando Navigator.
🧠analyze_agreementNavigator
Extrae cláusulas clave, identifica riesgos y genera insights con IA.
📋get_agreement_detailseSignature
Obtiene detalles completos de un sobre: firmantes, estatus, fechas.
✉️create_envelopeeSignature
Crea y envía un nuevo envelope desde template con destinatarios.
📄list_templateseSignature
Lista plantillas disponibles con filtro por nombre o categoría.
⚡trigger_maestro_workflowMaestro
Dispara un flujo de Maestro individual o bulk con parámetros dinámicos.

        "type": "oauth2",
        "scopes": ["signature", "impersonation"]
      }
    }
  }
